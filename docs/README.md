## ✏️ 기능 목록

- [ ] 자동차의 입력을 받아 저장
    - [X] 자동차 이름 입력 안내 메시지 콘솔에 출력
    - [ ] `,`를 기준으로 파싱해서 car 객체에 저장
      <br>

  #### 🎯예외사항
    - [ ] 파싱할 문자가 `,`가 아닐 경우
      `IllegalArgumentException` 발생
    - [ ] 자동차 이름이 0글자 or 5글자 초과일 경우
      `IllegalArgumentException` 발생
    - [ ] 자동차 이름이 중복된 경우
      `IllegalArgumentException` 발생
      <br><br><br>
- [ ] 시도할 회수를 입력받아 저장
    - [ ] 회수 입력 안내 메시지 콘솔에 출력
      <br>
  #### 🎯예외사항
    - [ ] 입력 값이 숫자가 아닐 경우
      `IllegalArgumentException`을 발생

<br>

- [ ] 한 번의 게임에 각 자동차의 이동 여부 확인
  <br>
  <br>
- [ ] 한 번의 게임이 진행될 때 마다 결과 출력
- [ ] 모든 게임이 끝난 후 결과 출력
